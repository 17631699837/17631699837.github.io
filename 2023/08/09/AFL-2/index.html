堆溢出和越界读取

1.构建并安装 libexif：

cd libexif-libexif-0_6_14-release/ sudo apt-get install autopoint libtool gettext libpopt-dev autoreconf -fvi ./configure --enable-shared=no --prefix="$HOME/fuzzing_libexif/install/" make make install

autoreconf -fvi 在软件项目中重新生成自动配置（Autoconf）和构建系统相关文件的命令

afl-clang-lto 使用了链接时间优化（Link-Time Optimization，简称LTO）的技术，可以在编译过程中对整个程序进行优化，提高模糊测试的效率和准确性。
